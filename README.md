TRIFECTA is a batch script that allows you to search a phrase or keywords with 12 different search engines. You can do so one at a time or all at once. The options are as follows:


1. Yahoo
2. Google
3. DuckDuckGo
4. Shodan
5. Swisscows
6. Bing
7. Carrot2
8. Thingful
9. Google Scholar
10. Wolfram Alpha
11. Metacrawler
12. Yandex
13. All Of The Above
